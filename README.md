# Food-Demand-Prediction

Year 3 Semester 1 Project
Author: **Tan Chin Rou, Lim Yihui, Tan Pei Yu, Hong Jia Xuan, Goh Yann See**

# Description
Food demand prediction project using XGBoost to forecast the next 10 weeks of orders and ARIMA/SARIMA to model onion prices. The forecasts are combined to estimate the required onion stock and expected cost.
